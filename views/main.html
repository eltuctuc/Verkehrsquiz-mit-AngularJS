Verkehrsquiz

Diese Minianwendung ist im Zuge meiner Bachelor-Arbeit entstanden und soll als Fragenquiz innerhalb eines Verkehrsparcours dienen. Dabei wird an einem Stativ eine Tablet angebracht. So kann der Fragenquiz mobil im Parcour eingesetzt werden.

Trotzdem des Einsatzes in einem Verkehrsparcour kann der Quiz auch über das Internet genutzt werden.

Im Quiz gibt es nur Ja-Nein-Fragen. Sie werden in einer zufälligen Reihenfolge angezeigt. Nach dem Beantworten wird kurz angezeigt, ob die Antwort richtig oder falsch war. Dann folgt eine neue Frage.

Zum Schluß kommt man zur Auswertung. Dort werden alle Fragen mit ihren richtigen Antworten angezeigt.
